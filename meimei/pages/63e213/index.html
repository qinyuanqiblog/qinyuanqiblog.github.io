# nvm 安装步骤说明

# window

GitHub地址：https://github.com/coreybutler/nvm-windows/releases (opens new window)

点击下载

同意协议

选择nvm安装位置，建议不要放在c盘，反正我放c盘，nvm 一直切换不过来，不知道什么鬼

选择nvm放nodejs的位置，这里也不建议放在c盘，反正会导致nvm一直选不上，不知道什么鬼

开始安装

安装成功，查看nvm设置

再把这个参数按照下文配置就好了

# nvm加速 把下面的代码复制到上面的settings.txt 后面就可以了

node_mirror:https://registry.npmmirror.com/mirrors/node/ (opens new window)

node_mirror:https://npmmirror.com/mirrors/node/ (opens new window)

npm_mirror:https://registry.npmmirror.com/mirrors/npm/ (opens new window)

npm_mirror:https://npmmirror.com/mirrors/npm/ (opens new window)

配置

root: E:\nvm

path: E:\nodejs

node_mirror:https://npmmirror.com/mirrors/node/ (opens new window)

npm_mirror:https://npmmirror.com/mirrors/npm/ (opens new window)

效果，淘宝换镜像了， 要改成上面那种

查看环境变量里面是否有下面的两个变量，有了就不用设置了， 没有就手动设置一下
