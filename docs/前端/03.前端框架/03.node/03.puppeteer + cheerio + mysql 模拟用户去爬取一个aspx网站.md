## 思路

- 思路1：模拟用户操作，像一个正常的用户去点击然后爬取数据，之后再把数据存储到数据库, 然后点击分页重复这个操作，直到爬取完所有数据
- 思路2：直接调用获取数据接口，然后解析数据，之后再把数据存储到数据库我们先用思路1的方式来做，
- 本来我想用思路2的做法来做，后来有个问题没解决，所以先按照思路1来实现
